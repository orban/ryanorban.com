## Summary

[Solve It With Code](/notes/solve-it-with-code/) is a 5-week online course teaching the SolveIt method — a structured approach to problem-solving that combines AI assistance with independent thinking and craftsmanship. Instructors are Jeremy Howard (fast.ai founder) and Eric Ries (Lean Startup author). The $500 course comes with 30 days of platform access and a cloud-based Linux environment integrating AI tools with Jupyter-style notebooks.

The course addresses a specific failure mode in AI-assisted development: using AI to generate code you don't understand, which produces projects that work initially but become unmaintainable. The SolveIt curriculum emphasizes understanding how solutions work, not just receiving them. The ten lessons span coding (data structures, algorithms, web programming with FastHTML), writing, reading academic papers, web APIs, system administration, web scraping, and startup building.

The preview cohort included engineers, a professor of English, a high school student, and physicists — an intentionally diverse group that suggests the method is domain-agnostic. The through-line is craftsmanship: iterative understanding over rapid generation. This is a direct response to \"vibe coding\" culture — a position that the correct use of AI tools involves more deliberate thinking, not less.

## Key points

- 5-week course on problem-solving method combining AI tools with independent thinking.
- Instructors: Jeremy Howard (fast.ai) and Eric Ries (Lean Startup).
- Addresses AI dependency and maintainability problems from blind code generation.
- Cloud-based Linux + Jupyter notebook platform with integrated AI tools.
- Domains: coding, writing, papers, APIs, sysadmin, scraping, startup building.
- $500, 30-day platform access. Intentionally cross-disciplinary cohort.
