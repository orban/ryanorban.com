## Summary

Spokenly is a speech-to-text application for macOS and iOS that transcribes directly into any application. The distinguishing feature: local AI models (Whisper, Parakeet) that run entirely on-device, so voice data never leaves your machine. 100+ language support with automatic language detection.

Agent mode enables hands-free control beyond dictation — you can search the web, launch apps, and run shortcuts without touching the keyboard. This extends the use case from pure transcription to voice-driven computing, similar in direction to Voice Control on macOS but with better accuracy and cross-app breadth.

The free tier covers local model usage (no cloud costs), with Pro at $9.99/month for cloud-enhanced features. 100,000+ active users and 4.9 App Store rating suggest genuine product-market fit. Compared to Handy (simpler, fully local) and Monologue (Every bundle, Mac/iOS focused), Spokenly sits in the middle — more capable than Handy, more standalone than Monologue.

## Key points

- Local models (Whisper, Parakeet) — voice data stays on device in local mode.
- 100+ languages with automatic detection.
- Agent mode: hands-free web search, app launch, shortcut execution.
- Real-time transcription with multiple speed/accuracy options.
- Free for local model usage; $9.99/month Pro for cloud features.
- 100,000+ users, 4.9 App Store rating.
