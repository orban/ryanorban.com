## Summary

[Readout](/notes/readout/) is a macOS application (currently in beta at v0.0.11) that consolidates your development environment into a single dashboard — replacing the fragmented state monitoring that typically requires six or more terminal tabs. The tagline is "your dev environment, at a glance."

The premise addresses a real friction point: running a typical development setup means juggling multiple terminal sessions for logs, processes, ports, database connections, test watchers, and more. Each lives in a separate tab with no unified view. [Readout](/notes/readout/) aims to give you a single pane of glass for the state of everything running locally.

The tool is early-stage (beta) but targeted at macOS 14+ (Tahoe). It's a focused, specific-purpose tool rather than a general-purpose terminal, which is the trend in developer tooling — alongside [cmux](/notes/cmux/) (multi-agent terminal), Baton (multi-agent dashboard), and similar tools that build on top of raw terminals to add awareness and organization.

## Key points

- Single dashboard replacing multiple terminal tabs for local dev environment state.
- macOS 14+ (Tahoe), currently v0.0.11 in beta — early but actively developed.
- Fits the pattern of purpose-built dev dashboards (alongside [cmux](/notes/cmux/), Baton) rather than general terminals.
