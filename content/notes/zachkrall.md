## Summary

Zach Krall's personal website is accessible via SSH (`ssh hi.zachkrall.com`) rather than a web browser — an intentional creative statement about the medium of a portfolio. The terminal is the interface; there's no web page to visit in a conventional sense.

This is an unusual and technically interesting choice. Running an interactive experience through SSH means the content is served as a terminal application (likely using a tool like Bubbletea, Charm, or a custom SSH server), and the developer/designer gets full control of the terminal UI without browser constraints. It's a niche technique that signals creative technical sensibility.

Zach Krall appears to work in creative technology and interactive design, where the distinction between portfolio site and art project is intentionally blurred.

## Key points

- Personal site at `ssh hi.zachkrall.com` — terminal as the interface, not a web browser.
- Creative technical choice: full SSH-served terminal experience rather than a conventional portfolio page.
- Suggests background in creative technology and interactive terminal design.
