## Summary

The Center for Land Economics released a suite of five free, open-source GIS tools for visualizing land value data geographically. The main argument from the article is that most people lack intuition about where economic value concentrates in cities — even urbanists who've read the theory. A 3D map showing Manhattan's assessed land values literally towering above the rest of New York City combined is more persuasive than any table or paragraph.

The flagship tool is CivicMapper, which renders 3D land value visualizations for nine cities. PIOAM (Put It On A Map) is the analysis-focused tool, offering selection-based statistics, parking lot detection via satellite imagery, comparable property searches, and time-adjusted price analysis — all running locally in the browser without uploading user data. Supporting tools handle the data pipeline: Data Fetcher extracts GIS data from government servers, Format Converter transforms between GIS file types, and File Constructor joins separate datasets.

The framing of just put it on a map is itself the insight. In land value taxation and urbanist advocacy, debates often stay abstract. These tools give advocates concrete geographic evidence they can generate themselves without expensive software licenses or GIS expertise. Similar patterns showing concentrated land value appear in Syracuse, Rochester, and Spokane — suggesting the Manhattan example isn't an outlier but a structural feature of urban land economics.

## Key points

- Manhattan's assessed land value exceeds the rest of New York City combined — visualizable with CivicMapper in seconds.
- Five tools covering the full pipeline: visualization, analysis, data extraction, format conversion, and dataset joining.
- All analysis runs locally in the browser — no data uploaded to external servers.
- Includes parking lot detection via satellite imagery, useful for illustrating underutilized land arguments.
- Land value taxation advocates can generate compelling visual evidence without GIS expertise or expensive software.
