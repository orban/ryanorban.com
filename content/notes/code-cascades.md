## Summary

[Code Cascades](/notes/code-cascades/) is a programming challenge platform organized into themed "cascades" — each cascade is a sequence of related tasks that start simple and build in complexity. Current themes include a 2D game challenge (medium difficulty, 5 tasks) and a digital gardening challenge (hard, 10 tasks). The idea is that each task in a cascade naturally extends from the previous one, creating a project-like arc rather than disconnected puzzles.

This positions [Code Cascades](/notes/code-cascades/) differently from LeetCode or HackerRank, which are primarily competitive/interview-prep platforms with independent problems. Code Cascades is more like a structured learning path — you build something incrementally, getting progressively harder requirements added to the same codebase or concept. This is closer to how real projects evolve than interview-style DSA problems.

The platform is early-stage and community-supported, with more cascades promised based on demand. The Reddit community provides help for stuck participants. The combination of project-based learning with a competitive element (leaderboards, completion tracking) tries to maintain motivation across longer SKILL-building arcs. Related to platforms like Advent of Code, Exercism, and Project Euler in the broader space of structured programming practice.

## Key points

- Themed cascades with escalating difficulty — tasks build on each other within a theme.
- Different model from LeetCode: project-like progression rather than disconnected puzzles.
- Current themes: 2D game development, digital gardening.
- Community support via subreddit; more cascades added based on interest.
- Suitable for intermediate learners building project experience, not competitive interview prep.
