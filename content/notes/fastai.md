## Summary

[fast.ai](/notes/fastai/) is Jeremy Howard and Rachel Thomas's open deep learning education project: a free course (Practical Deep Learning for Coders), an accompanying Python library (also called [fastai](/notes/fastai/)), and the book Deep Learning for Coders with fastai and PyTorch. The tagline "making neural nets uncool again" is a position against hype — the focus is practical implementation, not marketing.

The pedagogical philosophy is top-down: start by building working neural network applications (image classifiers, NLP models), then progressively understand the underlying math and mechanics. This is the opposite of the traditional academic approach (derive backpropagation from first principles before ever training a model). Jeremy Howard's argument is that motivation and intuition come from seeing things work, and the theory makes more sense once you've already used it.

[fast.ai](/notes/fastai/) has been widely credited with democratizing deep learning — making it accessible to practitioners without PhDs in mathematics. The course is free, uses PyTorch under the hood, and has been updated through multiple versions as the field has evolved. [fast.ai](/notes/fastai/) alumni include practitioners across ML engineering, research, and applications.

## Key points

- Free course, library, and book for practical deep learning in Python/PyTorch.
- Top-down teaching: use neural nets first, understand math later.
- Created by Jeremy Howard and Rachel Thomas — influential in democratizing ML education.
- [fastai](/notes/fastai/) library abstracts PyTorch for cleaner high-level code while remaining transparent.
- Regularly updated — covers modern techniques alongside foundations.
