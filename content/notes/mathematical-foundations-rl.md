## Summary

Mathematical Foundations of Reinforcement Learning is an open-access textbook by Shiyu Zhao (Westlake University) that covers reinforcement learning theory with mathematical rigor. Unlike applied RL courses that start with code and treat math as secondary, this book inverts the priority: the goal is a solid theoretical foundation before implementation.

The coverage follows the standard RL curriculum but with depth: Markov decision processes (MDPs), Bellman equations (both optimality and evaluation forms), value-based methods (Q-learning, SARSA), policy gradient methods, actor-critic algorithms, and temporal difference learning. The mathematical treatment makes the connections between these algorithms explicit — you can see why Q-learning is a special case of temporal difference learning, or how the policy gradient theorem derives from value function definitions.

The book is on GitHub as an evolving resource, which means it gets corrections and additions. It's positioned between the rigorous but dense Sutton and Barto (Reinforcement Learning: An Introduction) and more specialized resources. For someone already comfortable with linear algebra, probability theory, and basic dynamic programming, this provides the formal grounding that makes the field's zoo of algorithms coherent rather than a collection of tricks.

## Key points

- Rigorous mathematical treatment of reinforcement learning — theory-first, code-second.
- Covers: MDPs, Bellman equations, value functions, Q-learning, policy gradient, actor-critic.
- Makes algorithmic relationships explicit: shows why algorithms are special cases of general principles.
- By Shiyu Zhao (Westlake University) — free on GitHub, actively maintained.
- Best with prior background in linear algebra, probability theory, dynamic programming.
- Companion to Sutton and Barto for deeper mathematical grounding.
