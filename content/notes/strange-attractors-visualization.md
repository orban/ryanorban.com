## Summary

Shashank Tomar's blog post builds a Three.js particle system visualization of Strange Attractors and uses it to explain the underlying mathematics: dynamical systems, chaos theory, and the butterfly effect. The visual output is beautiful — particles tracing intricate geometric forms like the Lorenz attractor — and the explanation is accessible.

Strange Attractors are geometric objects in phase space that describe the long-run behavior of chaotic dynamical systems. A system's trajectory will orbit an attractor indefinitely but never exactly repeat — the paths are fractal in structure. The Lorenz attractor (shaped like a butterfly) is the canonical example, discovered when Edward Lorenz found that tiny changes in initial conditions produce radically different long-term outcomes.

The Three.js implementation uses a particle system where each particle traces the attractor's path in 3D space. This makes the geometry tangible in ways that 2D projections can't. The butterfly effect isn't just a metaphor — the post shows concretely how nearby trajectories diverge exponentially over time. Good example of using creative coding to make abstract mathematics visible and physically intuitive.
