## Summary

Terminal Trove is a curated directory for discovering terminal applications — the "$HOME of all things in the terminal." It catalogs TUI and CLI tools across categories: file managers, system monitors, git clients, text editors, database browsers, productivity tools, and more. For developers building their terminal environment, it's a systematic way to find well-maintained alternatives to whatever they're currently using.

The site addresses the discoverability problem with terminal tools. Great TUI apps like Lazygit, Htop, Ranger, Ncdu, and dozens of others exist but are scattered across GitHub, blog posts, and word of mouth. Terminal Trove aggregates them with descriptions and filtering, similar to what Awesome lists do but with more curation and a browsable interface.

As terminals have gotten more capable (better Unicode, true color, faster rendering), the TUI ecosystem has flourished. Ratatui, Bubbletea, and Textual provide high-quality TUI frameworks in Rust, Go, and Python respectively, making it easier to build polished terminal applications. Terminal Trove tracks this growing ecosystem.

## Key points

- Curated directory of CLI and TUI applications, browsable by category.
- Covers file managers, system monitors, git clients, database tools, productivity apps.
- Addresses the discoverability problem for terminal tooling.
- Complements Ratatui showcase and Awesome lists for finding terminal tools.
