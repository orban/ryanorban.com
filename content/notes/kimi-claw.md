## Summary

[Kimi Claw](/notes/kimi-claw/) is Moonshot AI's AI assistant platform, positioned as a 24/7 assistant with long-term memory and automation capabilities. The two highlighted features — persistent memory and automation — are the differentiators from standard chatbot interfaces.

The platform covers a range of productivity use cases: chat, deep research, document processing (websites, docs, slides, spreadsheets), code assistance via Kimi Code, and cross-platform access (Web, iOS, Android). The developer platform is open, suggesting integration capabilities beyond the consumer interfaces.

Moonshot AI is a well-funded Chinese AI company. Kimi is their primary consumer product. The \Claw\ branding appears to position this as an agentic tier of the Kimi assistant — not just answering questions but taking autonomous actions and maintaining memory across sessions. The 4.9-star rating across 94,000+ reviews indicates substantial usage traction.

## Key points

- Moonshot AI's AI assistant with long-term memory across conversations.
- Automation capabilities: not just Q&A but task execution.
- Deep research, document processing, Kimi Code for coding tasks.
- Cross-platform: Web, iOS, Android.
- Developer platform for integrations.
