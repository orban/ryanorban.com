## Summary

[Textream](/notes/textream/) is a free macOS app that turns a script into a live teleprompter, highlighting your current position in real time as you speak. It uses speech recognition to track where you are in the text, so the highlighting follows your actual words rather than running on a fixed timer. This matters for live presentations and recordings where pacing varies — a fixed-scroll teleprompter can't adapt if you pause or ad-lib, but [Textream](/notes/textream/) stays in sync with what you actually say.

The use cases are predictable but genuinely useful: live streams, podcast recordings, video interviews, presentations. The free price point and macOS-native design make it easy to adopt. No cloud dependency, no subscription — the speech recognition runs locally using macOS system capabilities.

Tools like this reflect how macOS system APIs have made certain categories of utility software easier to build. Speech recognition on Apple Silicon is fast enough and accurate enough that a solo developer can build a useful product on top of it in a weekend.

## Key points

- Real-time script highlighting synced to speech via speech recognition — not a fixed-scroll timer.
- Free macOS app with no cloud dependency.
- Targeted at live streams, podcast recordings, interviews, and presentations.
- Uses macOS built-in speech recognition APIs, runs locally.
- Clean single-purpose tool — does one thing and does it well.
