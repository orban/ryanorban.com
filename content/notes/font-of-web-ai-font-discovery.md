## Summary

Font of Web is an AI-powered tool for typography discovery and design inspiration. The core use case: describe the aesthetic you're going for or upload a reference design, and the tool suggests matching fonts with explanations for why they fit.

Font discovery is a well-known friction point in web design — there are hundreds of thousands of fonts available, the vocabulary for describing type is specialized, and browsing font libraries without knowing what you're looking for is tedious. AI assistance addresses this by letting you describe in natural language ("warm, editorial, slightly playful") and mapping that to specific typefaces.

The tool sits alongside resources like Google Fonts, Fontjoy (font pairing), and [Modern Font Stacks](/notes/modern-font-stacks/) (system font performance) as part of the web typography toolkit.

## Key points

- AI font matching: describe an aesthetic or upload a reference, get typeface recommendations.
- Reduces friction in font discovery — no need for typography vocabulary to search.
- Design inspiration alongside font suggestions.
- Complements Google Fonts, Fontjoy, and [Modern Font Stacks](/notes/modern-font-stacks/) in the type toolkit.
