## Summary

[AI Design Components](/notes/ai-design-components/) by ancoleman is a collection of 76 production-ready Claude skills organized across 10 development domains: frontend, backend, DevOps, infrastructure, security, cloud, AI/ML, data engineering, developer productivity, and FinOps. The skills provide Claude with domain expertise, decision frameworks, and production-ready code patterns for specific technical areas.

The 76 skills are packaged as 19 Claude Code plugins, installable via the Claude Code plugin marketplace or a shell installer. There's also a Skillchain v3.0 system — a guided workflow that chains multiple skills together for complex multi-domain tasks. This goes beyond individual skill invocation to orchestrating skills in sequence for common development scenarios.

The coverage is genuinely broad: the backend-AI-skills cover LLM integration patterns; the security skills cover OWASP, secrets management, threat modeling; the cloud-provider-skills cover AWS, GCP, Azure patterns. The FinOps category is unusual — cost optimization and cloud spend management as a skill domain. The documentation is at GitHub Pages, and the installer is interactive (menu-driven, not a single command).

## Key points

- 76 production-ready Claude skills across 10 domains (frontend, backend, DevOps, security, cloud, AI/ML, etc.).
- 19 plugin groups, installable via Claude Code plugin marketplace.
- Skillchain v3.0 system chains skills for complex multi-domain workflows.
- Covers unusual domains: FinOps (cloud cost optimization) and dedicated AI/ML patterns.
- MIT license; full documentation at GitHub Pages.
