## Summary

Paper is a design tool built around the premise that the gap between design and implementation is closing, and that AI agents should be in the loop throughout. The canvas exports directly to HTML/CSS code without translation — "nothing gets lost in translation" — because it's building on web standards rather than a proprietary format that needs to be interpreted by an engineer.

The agent integration is the differentiator from tools like Figma. You can connect IDEs, CLI tools, and AI assistants (Claude, GitHub Copilot, etc.) directly to the canvas for real-time collaboration. Agents handle responsive layouts, style variations, and repetitive design tasks. Live data from databases and APIs replaces placeholder text — designs show real content from the start.

Bidirectional sync between the canvas and the codebase means design tokens and components stay in sync. One source of truth across design and implementation. Available as web app and desktop app (macOS and Windows).

## Key points

- Canvas exports as HTML/CSS directly — no designer-to-developer translation step.
- Connects Claude, GitHub Copilot, and other AI agents to the canvas for real-time design collaboration.
- Live data from databases and APIs — no placeholder content in designs.
- Bidirectional sync: design tokens and components update between canvas and codebase.
- Positions as the design tool for teams shipping with agents, not just designing for humans to implement.
