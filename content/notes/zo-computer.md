## Summary

Zo is a personal cloud computer that runs AI agents autonomously 24/7. You send text instructions in natural language; it executes continuously — including while you sleep. The pitch is a unified personal workspace that replaces a stack of SaaS tools with a single AI-driven cloud environment you fully control.

The capabilities are broad: email and calendar management, building apps and websites, file handling across most formats (PDF, spreadsheets, code, audio, video), web hosting, and API integrations. You can select from multiple LLM providers, and the system has built-in transcription, image generation, and video creation. The AI operates with full file system access, so it can take on complex multi-step workflows without needing explicit tool configuration.

The architectural bet is \persistent autonomy\ — regular snapshots for state restoration, continuously running agents, managed workflows. This is closer to a personal cloud desktop than a chatbot. The model is: you own the environment, the AI has full access to it, and work happens asynchronously rather than interactively. Similar in spirit to [Nightshift](/notes/nightshift/) (run agents autonomously overnight) but as a full cloud platform rather than a codebase maintenance tool.

## Key points

- Personal cloud computer with 24/7 autonomous AI agent execution via natural language.
- Full file access across formats (PDF, code, audio, video, spreadsheets).
- Manages email, calendar, builds apps, handles workflows.
- Multi-model support with built-in image generation, transcription, video creation.
- Persistent autonomy: regular snapshots, continuous execution, no need to be online.
- Positioned as a SaaS consolidator — one workspace instead of many tools.
