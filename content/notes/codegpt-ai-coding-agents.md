## Summary

CodeGPT is an AI coding assistant available as extensions for VS Code and JetBrains IDEs (IntelliJ, PyCharm, etc.) that operates on a Bring Your Own Key (BYOK) model. You supply your own API keys for Claude, Gemini, or other providers — your code stays under your control and isn't metered by CodeGPT. The company's revenue comes from improving the product, not tracking API usage.

The differentiation from GitHub Copilot and similar tools is model flexibility and transparency. Rather than being locked to one underlying model, you can switch between Claude, Gemini, and other providers at will. The AI shows its reasoning before applying changes — you see the thinking process, not just the result. MCP integrations, customizable rules, and live context tracking extend the core assistant.

The AI agent approach goes beyond autocomplete: CodeGPT's agents can plan multi-step solutions, reason about codebases, and execute changes through a three-step process (think, plan, build). For developers who want the capability of frontier LLMs applied to coding without vendor lock-in or opaque data handling, this is the relevant alternative.

## Key points

- BYOK model: bring your own API keys (Claude, Gemini, others) — no vendor lock-in.
- Available in VS Code and JetBrains IDEs.
- Shows AI reasoning before applying changes — transparency over magic.
- AI agent mode: plans and reasons, not just autocomplete.
- MCP integrations for extending context and tool access.
- Privacy-first: code doesn't leave your control unless you explicitly authorize it.
